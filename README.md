# Tabular Editor Scripts & Tools

This repository contains custom scripts and documentation for Tabular Editor (TE2), particularly for working with Power BI models.

## Repository Contents

- `CustomActions.json`: Custom actions configured for Tabular Editor.
- `TE2_SelectiveExport.cs`: C# Script for selective exporting of objects.
- `TE2_SelectiveImport.cs`: C# Script for selective importing of objects.
- `PBI_RoleCopy_HowTo.docx` & `PBI_RoleCopy_HowTo.pdf`: Documentation guides on copying roles in Power BI using Tabular Editor.
